## 🇹🇷 Türkçe

### 📖 Hakkında

**Barly Browser**, Electron üzerine inşa edilmiş modern ve hafif bir masaüstü tarayıcıdır. 25'ten fazla yerleşik özellik ile geliştirici dostu bir deneyim sunar. Tamamen Türkçe arayüze sahiptir.

### ✨ Özellikler

| 🔧 Kategori | 📋 Özellikler |
|:---:|---|
| 🌍 **Tarayıcı** | Sekmeli gezinti, yan yana görünüm, gizli mod, QR kod oluşturucu |
| 🤖 **AI Asistan** | 10 farklı AI servisi (Gemini, ChatGPT, Claude, Copilot, Grok, DeepSeek, Mistral, Perplexity, You.com, Phind), sekmeli AI paneli, yan yana AI modu |
| 🎨 **Temalar** | Koyu/Açık mod, özel tema oluşturucu, hazır tema paketleri, duvar kağıdı desteği |

| 🕐 **Geçmiş** | Arama geçmişi, günlük kullanım istatistikleri, zaman takibi |
| 📝 **Notlar** | Yerleşik not defteri, yapılacaklar listesi, web üzerinde açıklama ekleme |
| 🔐 **Güvenlik** | Şifre yöneticisi, HTTPS uyarıları, reklam engelleyici |
| 📌 **Oturumlar** | Oturum kaydetme/yükleme, otomatik doldurma |
| ⌨️ **Kısayollar** | Özelleştirilebilir klavye kısayolları |
| 📴 **Çevrimdışı** | Sayfaları çevrimdışı kaydetme |
| ⏱️ **Zaman Limiti** | Site bazlı kullanım süresi sınırlandırma |
| 🎵 **Mini Player** | Arka planda müzik/video kontrolü |

| 🖥️ **Geliştirici Konsolu** | Yerleşik DevTools erişimi |

### 🚀 Kurulum

```bash
# Repoyu klonla / Clone the repo
git clone https://github.com/Barand1500/Barly-Browser.git

# Klasöre gir / Enter directory
cd Barly-Browser

# Bağımlılıkları yükle / Install dependencies
npm install

# Çalıştır / Run
npm start
```

## 🇬🇧 English

### 📖 About

**Barly Browser** is a modern, lightweight desktop browser built on Electron. It offers a developer-friendly experience with 25+ built-in features. The interface is fully in Turkish with intuitive icons.

### ✨ Features

| 🔧 Category | 📋 Features |
|:---:|---|
| 🌍 **Browser** | Tabbed browsing, split view, incognito mode, QR code generator |
| 🤖 **AI Assistant** | 10 AI services (Gemini, ChatGPT, Claude, Copilot, Grok, DeepSeek, Mistral, Perplexity, You.com, Phind), tabbed AI panel, side-by-side AI mode |
| 🎨 **Themes** | Dark/Light mode, custom theme builder, preset themes, wallpaper support |
| 📑 **Bookmarks** | Bookmark manager, quick access panel, folder structure |
| 🕐 **History** | Search history, daily usage statistics, time tracking |
| 📝 **Notes** | Built-in notepad, to-do list, web annotations |
| 🔐 **Security** | Password manager, HTTPS warnings, ad blocker |
| 📌 **Sessions** | Session save/load, autofill |
| ⌨️ **Shortcuts** | Customizable keyboard shortcuts |
| 📴 **Offline** | Save pages for offline reading |
| ⏱️ **Time Limits** | Per-site usage time limiting |
| 🎵 **Mini Player** | Background media control |
| 🔗 **Link Preview** | Hover preview for links |
| 🔍 **Find in Page** | Quick search with Ctrl+F |
| 📸 **Screenshot** | One-click page capture |
| 🖥️ **Dev Console** | Built-in DevTools access |

### ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + T` | New Tab |
| `Ctrl + W` | Close Tab |
| `Ctrl + L` | Address Bar |
| `Ctrl + F` | Find in Page |
| `Ctrl + Shift + I` | Developer Tools |
| `Ctrl + D` | Add Bookmark |
| `F5` | Refresh Page |
| `F11` | Fullscreen |
